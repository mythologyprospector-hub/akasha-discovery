# akasha-discovery

The discovery engine of the Akasha ecosystem.

This repository explores unknown structures by examining the
knowledge defined in **akasha-world** and identifying potential gaps,
patterns, and unexplained relationships.

Discovery outputs **candidates**, not truth.

All candidates must be reviewed by a human steward before integration
into the canonical knowledge base.

---

## Role in the Akasha System

akasha-world → discovery → candidate → human review → world update

Discovery never writes directly into the canonical world registry.

Human review is required for all accepted discoveries.

---

## Responsibilities

- detect structural gaps
- propose candidate entities
- propose candidate relationships
- explore hypothesis spaces
- record experiments

---

## Dependencies

- akasha-axioms
- akasha-world

---

## Outputs

- candidate discoveries
- experiment results
- hypothesis records
